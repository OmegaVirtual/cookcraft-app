# CookCraft 🍽️

A full-stack recipe and nutrition web application built using .NET, designed to help users manage recipes, track allergens, and generate smart grocery lists.

---

## 🚀 Features

- 🔍 Search and filter recipes  
- 🌐 Import recipes from external websites  
- ⚠️ Allergen detection system  
- 🛒 Smart grocery list generator  
- 👤 User profiles and authentication  
- 📱 Responsive UI  

---

## 🛠️ Tech Stack

- **Backend:** ASP.NET Core (.NET, C#)  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQL  
- **Hosting:** Cloudflare  

---

## 💡 Project Highlights

- Built independently as a full-stack application  
- Designed using MVC architecture  
- Implemented real-world features focused on usability  
- Handles dynamic user data and interactions  

---

## 📂 Project Structure

- `Controllers/` – application logic  
- `Models/` – data models  
- `Views/` – UI rendering  
- `wwwroot/` – static files
